# 📊 ETL de Controle de Estoque com KNIME

## Visão Geral

Este projeto demonstra a construção de um processo ETL (Extração, Transformação e Carga) utilizando a plataforma KNIME Analytics Platform.

O fluxo realiza a importação de dados de estoque a partir de uma planilha Excel, executa etapas de limpeza e transformação dos dados, aplica regras de negócio e exporta o resultado final para uma nova planilha.
